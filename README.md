
# Profile Tracker



## Installation
Please follow below steps to run the project locally.

**Step 1 :-**  
clone the project from github

```bash
  git clone https://github.com/MdMehedi-Hasan/profile-tracker.git
```

**Step 2 :-**  
Run Frontend

```bash
  npm install
  npm run dev 
```
> **_NOTE:_**  You might need to install additional npm packages.
